# 牛子PK

群聊小游戏: 牛子PK

### 指令表

以下命令可省略前缀'/

| **指令**                     | **说明**                                         |
|-----------------------------|------------------------------------------------|
| `/开启牛子秘境`              | 开启插件，并在当前群聊启用牛子PK                |
| `/关闭牛子秘境`              | 关闭插件，并清除所有群聊的开关状态             |
| `/牛子帮助`                  | 查看帮助                                        |
| `/启用牛子pk`                | 在群聊中启用牛子PK                             |
| `/禁用牛子pk`                | 在群聊中禁用牛子PK                             |
| `/牛子`                      | 查看自己的牛子信息                             |
| `/pk @用户`                  | 对at的用户进行牛子PK，一次只能PK一个用户       |
| `/🔒(/suo/嗦/锁)我`          | 嗦自己的牛子                                    |
| `/🔒(/suo/嗦/锁) @用户`       | 嗦at用户的牛子                                  |
| `/打胶`                      | 安抚自己的牛子                                  |
| `/看他牛子(/看看牛子) @用户`    | 查看at用户的牛子                               |
| `/注册牛子`                  | 注册属于你的牛子                                |
| `/牛子排名(/牛子排行)`         | 查看牛子排行榜                                  |
| `/牛友(/牛子好友/牛子朋友)`   | 查看牛子好友                                    |
| `/关注牛子(/添加牛友)`         | 关注牛子好友                                    |
| `/取关牛子(/删除牛友)`         | 取关牛子好友                                    |
| `/牛子转生`                  | 进行牛子转生                                    |
| `/牛子成就`                  | 查看牛子成就                                    |
| `/牛子仙境`                  | 进入牛子仙境                                    |
| `/牛子修炼(/牛子练功/牛子修仙)` | 进行牛子修炼                                    |
