# nonebot-plugin-withdraw
## 简单撤回插件，让机器人撤回 **自己发出的消息**
### 使用

**以下命令需要加[命令前缀](https://v2.nonebot.dev/docs/api/config#Config-command_start) (默认为`/`)，可自行设置为空**

#### 方式1：

`@机器人 撤回 [num1][-num2]`

*注：若命令前缀为空则需要 @机器人，否则可不@*

`num` 指机器人发的倒数第几条消息，从 `0` 开始，默认为 `0`

`num`不需要加`[]`，仅为了说明是可选参数

示例：

```
@机器人 撤回    # 撤回倒数第一条消息
@机器人 撤回 1    # 撤回倒数第二条消息
@机器人 撤回 0-3    # 撤回倒数三条消息
```

#### 方式2：

回复需要撤回的消息，回复“撤回”
