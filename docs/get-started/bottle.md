# 漂流瓶

扔漂流瓶到大海~

## 用户手册

- `扔漂流瓶` [文本/图片] 或 `扔漂流瓶` 后一条消息跟漂流瓶内容
- `寄漂流瓶` [文本/图片] （同`扔漂流瓶`，防止指令冲突用）
- `捡漂流瓶`
- `查看漂流瓶` [漂流瓶编号]
- `点赞漂流瓶` [漂流瓶编号]
- `评论漂流瓶` [漂流瓶编号] [文本]
- `举报漂流瓶` [漂流瓶编号]
- `删除漂流瓶` [漂流瓶编号]
- `我的漂流瓶`

## 管理员手册

- `清空漂流瓶`
- `恢复漂流瓶` [漂流瓶编号]
- `删除漂流瓶评论` [漂流瓶编号] [QQ号]
- `漂流瓶白名单` [QQ / 群聊] [QQ号 / 群号]
- `漂流瓶黑名单` [QQ / 群聊 / 举报] [QQ号 / 群号]
- `漂流瓶详情` [漂流瓶编号]
